# Introduction

RHM is a Furniture E-Commerce project built with vanilla JS and attractive animated UI

**visit our website [Here](https://rhm-e-commerce.web.app)**

## Technologies
- HTML
- CSS
- JS
- Bootstrap

## Features

- Authentication
    - Login
    - Register
    - Logout
- Product management.
- Category management (Filteration).
- Search
- Profile (Favorites, Order history)
- Order management
- Contact us
- Lazy loading
